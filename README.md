# How To Become a Purveyor of Humor

1. **Fork** this repository by clicking the "Fork" button in the top right.
2. **Clone** your fork to your machine.
3. **Create** a new `.txt` file containing your funniest joke and add it to your fork.
4. **Save** your changes using `git add`, `git commit`, and `git push`. [Git cheat sheet](https://git-scm.com/cheat-sheet)
4. **Contribute** to the original repository by opening a pull request.
